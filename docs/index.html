🛡️ Is it safe? Yes — and you can verify it

Prism handles your Roblox login, so trust matters. That's exactly why it's open source: read every line, or build it yourself.

  • Your cookie never leaves your PC. It's stored locally, encrypted with Windows DPAPI (tied to your user account), and only ever sent to Roblox's own official login API to start the game.
  • No telemetry, no servers. Prism doesn't phone home. There's no backend that could collect anything.
  • 100% open source (MIT). The full source is on GitHub — audit it or compile it yourself and compare.
  • About antivirus flags: Prism is an unsigned .NET app that reads your Roblox cookie and updates itself, which some heuristic scanners false-flag. Microsoft Defender and the vast majority of engines read it clean. When in doubt, build it yourself below.

Build it yourself

Don't trust a binary? Compile Prism from source in two commands and get the exact same app.

You'll need: Windows 10/11, the .NET 8 SDK, and the WebView2 runtime (already on most Windows 11) to run it.

  1. Clone the repo: 
    git clone https://github.com/Cosmic4796/Prism.git
cd Prism
  2. Publish a self-contained single-file build: 
    dotnet publish -c Release -r win-x64 --self-contained true ^
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
  3. Your Prism.exe is here: 
    bin\Release\net8.0-windows\win-x64\publish\Prism.exe

That's the same build process used for the official release — so you can confirm the download matches the source.
