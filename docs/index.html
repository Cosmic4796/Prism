Prism

A free, open-source Roblox account manager. Run multiple accounts at once and launch all your alts in one click.

Download GitHub Discord

Features

  • Run multiple Roblox accounts at the same time (multi-instance)
  • Launch all your selected accounts in one click
  • Join the same server with every account
  • Works with Bloxstrap, Fishstrap, Froststrap, or the official client
  • Log in with Roblox or paste a cookie — no passwords stored
  • Auto-updates from inside the app

Is it safe?

Prism handles your Roblox login, so it's open source — you can read every line or build it yourself.

  • Your cookie stays on your PC, encrypted with Windows DPAPI. It's only ever sent to Roblox's own login API to start the game.
  • No telemetry and no servers — Prism doesn't phone home.
  • It's unsigned, so some antivirus engines may show a heuristic false positive. If unsure, build it yourself below.

Build it yourself

Needs Windows, the .NET 8 SDK, and the WebView2 runtime to run.

git clone https://github.com/Cosmic4796/Prism.git
cd Prism
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true

The exe ends up at bin\Release\net8.0-windows\win-x64\publish\Prism.exe — the same build the release uses.
